ImageMapTool

  1. Enter the image URL and click the Load Image button.
  2. Click the New Link button to create a new point area.
  3. Enter image URL and image link description.
  4. Drag image map above the image to the desired location.
  5. Map area size can be set by dragging the box corner.
  6. Click the Make Code button to generate map code.
  7. You can use the code right away or preview the linked contents by clicking the Preview button.
